JimLang

简单 · 实用 · 可嵌入 的 JVM 脚本语言

REPL · ANTLR 语法 · JSR‑223 集成 · 可与 Java 无缝互操作

快速开始 查看特性 GitHub

核心特性

🧮

语言内核

变量、函数、表达式求值，控制流（if/else、while、for、break/continue），链式运算。

💬

REPL 交互

可保留状态，支持 :vars/:funcs/:clear/:help/:exit 命令。

🧰

标准库

字符串（len/upper/lower/trim/substring/indexOf/split）、数学（max/min/abs/round/random）、文件（read/write/exists）。

🧩

JSR‑223

作为脚本引擎嵌入 Java：ScriptEngine engine = manager.getEngineByName("jim");

⚙️

ANTLR 4

采用 ANTLR 4 定义语法，易于扩展与维护。

📦

可集成

Maven 构建与测试，示例与文档完善，易于在项目中集成使用。

快速开始

REPL

mvn -q exec:java -Dexec.mainClass="com.dafei1288.jimlang.Repl"
JimLang> println("Hello, JimLang!")
Hello, JimLang!

Java 原生 API

import com.dafei1288.jimlang.JimLangShell;

JimLangShell shell = new JimLangShell();
String script = """
  function add(a, b) { return a + b }
  println(add(5, 3))
""";
Object result = shell.eval(script, null);

JSR‑223

import javax.script.*;

ScriptEngine engine = new ScriptEngineManager().getEngineByName("jim");
engine.eval("println('Hello from JSR-223!')");

示例

Hello World

println("Hello from JimLang!")

函数与控制流

function factorial(n) {
  if (n <= 1) { return 1 }
  return n * factorial(n - 1)
}
println("5! = " + factorial(5))

数组与对象

var arr = [1, 2, 3]
println(arr[0])

  • 构建：mvn clean package；仅编译：mvn clean compile
